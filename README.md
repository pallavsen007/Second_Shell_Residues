# Second_Shell_Residues
Supporting files for the paper describing gates in MtbThyX

# Packages used in the paper
- GROMACS 2023(https://www.gromacs.org/)
- PyMol(https://pymol.org/)
- VMD(https://tcbg.illinois.edu/Research/vmd/)
- pmx(https://github.com/deGrootLab/pmx)
- alchemlyb(https://alchemlyb.readthedocs.io/en/latest/)

## Directory Structure

- complex_state: contains the hybrid structures and topologies of complex states with corresponding mdp file to do the calculations for the transformations
- free_state: contains the hybrid structures and topologies of free states with corresponding mdp file to do the calculations for the transformations
- structures: starting structures of complex and free states
